使用Hexo搭建博客

一直想自己实现一个博客网站，一开始购买阿里云服务器也是如此。最初使用的事WordPress，博客存活了不到一星期我就重装系统了（可能对于我来说没有经过新手村就直接打怪不舒服）。现在工作了偶然间看到我的部长弄得博客于是乎前去请教了一番，才了解到Hexo。下面说说我这周末的自己的一些对Hexo的搭建的了解。

准备阶段

  • NodeJs
  • Git

    声明一下，上述两个网站是官网。我并没有直接点击下载。关于nodejs我是使用的nvm下载的，当然也可以使用n下载(两者直接百度都可以下载)。git使用官网下载很慢，我使用的是淘宝源代理下载。传送门

安装Hexo

上传github

  • 新建仓库（注意你的仓库名字是[github.username].github.io）
  • 打开myhexo文件夹下的_config.yml文件修改以下内容
  • 发布到github
  • 这样去你自己的仓库下面就会看到这个东西，现在你输入http://[github.username].github.io,就可以看到你的博客了，跟本地4000端口看到的一样。

github上绑定自己的域名

  • 点击仓库上的setting按钮

  • 在图示地方输入你的域名（域名百度一搜自己买吧，学生有优惠）

  • 域名配置，我买的阿里云的，我就拿阿里云的举例子了，找到你买的域名，点击解析，小白的话点击那个新手操作会让你输入IP地址，这个IP地址是你的github上项目的IP地址，摁住win+r输入cmd打开终端，输入ping [github.username].github.io,出现的那一串数字就是我们要的ip地址。点击确定等待十分钟输入自己域名就可以查看自己的博客了~~
